# Ibn El-Akaber – Premium Men's Fashion Website

## Overview

Ibn El-Akaber is a modern luxury fashion website designed for premium men's clothing brands. The project focuses on delivering a professional shopping experience through elegant design, responsive layouts, and interactive front-end functionality.

Built using pure HTML, CSS, and JavaScript, the website demonstrates modern front-end development practices and serves as a strong foundation for future full-stack e-commerce expansion.

---

## Features

### Responsive Design

* Mobile-first approach
* Optimized for desktop, tablet, and mobile devices
* Adaptive navigation menu

### Product Showcase

* Dynamic product rendering using JavaScript
* Product cards with pricing and stock information
* Quick-view image previews

### Shopping Cart

* Add products to cart
* Quantity management
* Remove items from cart
* Persistent cart storage using Local Storage

### WhatsApp Checkout

* Automatic order generation
* Direct WhatsApp integration
* Easy customer communication

### Interactive UI Components

* Lightbox gallery
* Testimonials carousel
* Scroll reveal animations
* Smooth navigation

### Contact Section

* Contact form
* WhatsApp communication
* Email integration
* Direct phone access
